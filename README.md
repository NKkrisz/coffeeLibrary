# CoffeeLibrary - Book library management made with Java

## Functionality:

- ### Book management
    - Add books
    - Remove books
    - Edit details
    - Search books by title

- ### Borrower management (users)
    - Add borrowers
    - Remove borrowers
    - Edit borrower details

- ### Loan management
  - Add loans
  - Remove loans
  - Edit loan details

#### By default there are some example values inserted
